# Summary

`Org and Reference Resources` is a selective ingest of the folder originally labeled `Dont know what that is, Heres Who to know`. The durable value in this set is organization reference material, acronym decoding, PM-to-subcontractor expectation guidance, and a few practical operating tips for new PMs.

## Key Takeaways

- The acronym list is a useful translation layer for Graydaze-specific shorthand and construction vocabulary.
- `Who we are and What we do` maps named team roles and responsibilities across startup, CX/EX, finance, travel, and mission support functions.
- `WHAT TO EXPECT FROM PM SUBS` makes the PM-sub relationship more explicit from both sides.
- `New PM Tips` reinforces practical customer, tenant, and foreman communication habits.
