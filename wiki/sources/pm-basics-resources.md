# Summary

`PM Basics Resources` is a mixed reference folder that reinforces Graydaze operating principles, role expectations, travel and card administration, hotel and product reference data, and a few duplicate training assets. It reads like the working desk-drawer for a PM rather than a single coherent course packet.

## Key Takeaways

- The `12-Step` and `Behaviors` documents define the cultural and managerial style expected of a Graydaze PM.
- The credit-card coding sheet, travel info, hotel codes, and truck allowance policy support everyday administrative execution.
- Product and vertical-joint reference assets reinforce technical field readiness.
- Several files duplicate dedicated source groups already ingested, including Epicor, travel, and paint basics material.
