# Current Thesis

This vault is a compiled second brain. Raw sources live under `raw/` as immutable inputs. Durable knowledge lives under `wiki/` as maintained markdown pages. `AGENTS.md` defines how the agent ingests, queries, lints, and maintains the system.

## Structure

- `raw/sources/` holds immutable source material.
- `raw/assets/` holds local images and attachments referenced by sources.
- `wiki/sources/` holds maintained pages for ingested sources.
- `wiki/concepts/` holds evolving topic pages.
- `wiki/entities/` will hold named entities when they become worth tracking.
- `wiki/syntheses/` holds higher-level analyses and evolving theses.
- `wiki/queries/` will hold durable answers worth revisiting.
