# Current Synthesis

GC Pay is the Graydaze system used for subcontractor setup, SOV creation, change-order management, and pay-app workflows. In this corpus it appears as a core PM responsibility rather than an isolated back-office tool.

## Main Uses

- create or validate project setup and crew assignment
- create and upload SOVs
- create contractor change orders for single-line or multi-line changes
- manage subcontractor pay-app timing and approvals
- support project handoff and mission-support coordination
