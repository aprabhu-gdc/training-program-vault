# Current Synthesis

Ramp is the company-card system used for expense spending, coding, and receipt accountability. In this corpus it is framed as both a practical finance tool and a behavioral accountability tool tied to Graydaze values.

## Main Uses

- activate and register company cards
- code purchases promptly and correctly
- retain receipts and support accountability
- connect travel-related workflows and notifications
